## Installation
---

## With pip

    # Install package from pypi
    $ pip3 install Noty
    
    # Run it
    $ Noty
    
    # You are now ready to use it!

## Without using pip
    
    # Clone the repository
    $ git clone https://github.com/GrgBls/Noty.git
    $ cd Noty
    
    # Install Noty
    $ sudo python3 setup.py install
    
    # Run it
    $ Noty
    
    # You are now ready to use it!
    
## How to:
    
    # Run it
    $ Noty
    
    #Type your notes
    $ Remember to buy milk.
    
    #You are now ready to use it.
