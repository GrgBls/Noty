## Installation
---

### With pip

    # Install package from pypi
    $ pip install Noty
    
    # Run it
    $ Noty
    
    # You are now ready to use it!

### Without using pip
    
    # Clone the repository
    $ git clone https://github.com/GrgBls/Noty.git
    $ cd Noty
    
    # Install Noty
    $ sudo python setup.py install
    
    # Run it
    $ Noty
    
    # You are now ready to use it!
